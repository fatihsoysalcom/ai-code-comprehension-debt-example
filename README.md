# AI Code Comprehension Debt Example

This example demonstrates 'Comprehension Debt' in code, a concept where functional code is difficult to understand or maintain. It presents two Python functions: one simulating AI-generated code with high debt, and another refactored version with low debt, performing the same data processing task.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
